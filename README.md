# SpecTaroscoPy
Open-source application for spectral data analysis (photoelectron spectroscopy).

こんにちは。分子研の中澤遼太郎です。
こちらはスぺクトル解析を行うツール ”SpecTaroscoPy” です。

ライセンスはCC BY-NC 4.0 としました。
ご理解ください。
